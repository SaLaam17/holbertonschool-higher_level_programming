# Description:
'MyList': A custom list class that extends the built-in list class.
Includes a method 'print_sorted' that prints the list, but sorted in ascending order.

# Loading the class:
>>> MyList = __import__('1-my_list').MyList

# Example usage:

# Creation of a MyList instance and adding elements
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)

# Print the original list
>>> print(my_list)
[1, 4, 2, 3, 5]

# Call the print_sorted method
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

# Ensure the original list remains unchanged
>>> print(my_list)
[1, 4, 2, 3, 5]
